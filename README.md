# GitHubApp
Android app that enables the user to search GitHub repositories and interact with them.

Developed using the latest technologies:
  1. Jetpack Compose
  2. MVVM
  3. Repository pattern
  4. Retrofit
  5. Koin DI
  6. Compose Destinations
